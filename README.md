## Simplified Blog System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/53d8d3a5e80940718377264c028496d8/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Blog Information
5. Forum
6. Announcement Information
7. Personal Center
8. Background Management (Homepage, Personal Center, Blog Information Management, Forum Management, Announcement Information Management)


**Administrator Features:**
1. Login
2. Homepage
3. Personal Center
4. Administrator Management
5. Blog Information Management (Blog Information Management, Blog Comment Management, Blog Collection Management)
6. Basic Data Management (Blog Type Management, Post Type Management, Announcement Type Management)
7. Forum Management
8. Announcement Information Management
9. User Management
